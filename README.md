# Experiment 67: LinkedList Traversal

## Problem Statement
Create a `LinkedList` of integers. Use a `ListIterator` to traverse the list in forward direction and then in backward direction.

## Input Format
* **Line 1:** N.
* **Line 2:** N integers.

## Output Format
* **Line 1:** Forward: [elements]
* **Line 2:** Backward: [elements]

## Example 1

**Input:**
```text
3
10 20 30
```

**Output:**
```text
Forward: 10 20 30
Backward: 30 20 10
```
